Welcome to the Response Onset Tool

This tool, written in Matlab, helps you process your audiofiles from your experiment. Essentially, it tells you when the subjects verbal response to a stimulus sets in. With this Matlab tool, we aim to help others determine the response onset in their experiment data (audio).

About the project

Originally, we only looked toward an easy way to help us determine the verbal response onset in an audio file. However, as we looked to automate as much of the process as possible, things quickly became more involved. Additionally, we realised the help this tool might provide to others in the same situation.
